## Requirements

- Node 6.13.0
- Docker

## Installation
```sh
npm install
```

## Development
```sh
npm server-dev
```

## Deployment
### Build a new local image
```sh
npm run docker:build
```
### Push the image to Docker Hub
```sh
docker push maxmilpro/proxy
```

### Run a local container
```sh
npm run docker:run
```
